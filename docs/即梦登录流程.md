本文档记录了即梦 AI 视频生成平台的自动化登录流程，包含手动操作步骤和 Playwright 自动化脚本。

## 一、手动登录流程

| 步骤 | 操作 | 页面状态 |
|------|------|----------|
| 1 | 打开即梦官网 | `https://jimeng.jianying.com/` |
| 2 | 点击"开启即梦"按钮 | 进入创作页 |
| 3 | 处理登录协议弹窗 | 点击"同意"按钮 |
| 4 | 跳转抖音授权页 | 显示抖音扫码二维码 |
| 5 | 用户使用抖音 APP 扫码授权 | 页面自动跳转 |
| 6 | 登录成功 | 进入即梦创作界面 |

## 二、Playwright 自动化登录代码

```python
from playwright.sync_api import sync_playwright

def login_jimeng():
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)
        context = browser.new_context(viewport={'width': 1280, 'height': 720})
        page = context.new_page()
        
        # 步骤 1：打开即梦官网
        page.goto("https://jimeng.jianying.com/")
        page.wait_for_load_state("networkidle")
        print("✅ 已打开即梦官网")
        
        # 步骤 2：点击"开启即梦"按钮
        # 注意：按钮文字可能是"开启即梦"或"开始创作"
        start_button = page.locator('text=开启即梦').or_(page.locator('text=开始创作'))
        start_button.click()
        page.wait_for_timeout(2000)
        print("✅ 已点击开启按钮")
        
        # 步骤 3：处理登录协议弹窗（如果有）
        try:
            agree_button = page.locator('text=同意').first
            if agree_button.is_visible(timeout=3000):
                agree_button.click()
                print("✅ 已点击同意协议")
        except:
            print("ℹ️ 没有协议弹窗，继续")
        
        # 步骤 4：等待跳转到抖音授权页
        page.wait_for_url("**/open.douyin.com/platform/oauth/**", timeout=10000)
        print("✅ 已进入抖音授权页")
        
        # 步骤 5：截图保存二维码供用户扫描
        qr_code = page.locator('img[class*="qrcode"]').or_(page.locator('.qr-content img'))
        if qr_code.is_visible(timeout=5000):
            qr_code.screenshot(path="douyin_qrcode.png")
            print("✅ 二维码已保存到 douyin_qrcode.png")
            print("📱 请使用抖音 APP 扫描二维码完成授权")
        
        # 步骤 6：等待用户授权完成（轮询检测）
        print("⏳ 等待用户扫码授权...")
        
        max_wait = 120  # 最多等待 120 秒
        for i in range(max_wait):
            page.wait_for_timeout(1000)
            
            # 检查是否已跳转回即梦
            if "jimeng.jianying.com" in page.url and "generate" in page.url:
                print(f"✅ 登录成功！当前页面: {page.url}")
                break
            
            # 每 10 秒刷新一次页面状态
            if i % 10 == 0:
                print(f"⏳ 等待中... ({i}/{max_wait}秒)")
        else:
            print("❌ 登录超时，请重试")
            return False
        
        # 步骤 7：保存登录状态供后续使用
        context.storage_state(path="jimeng_auth.json")
        print("💾 登录状态已保存到 jimeng_auth.json")
        
        browser.close()
        return True

if __name__ == "__main__":
    login_jimeng()
```

## 三、复用已保存的登录状态

```python
from playwright.sync_api import sync_playwright

def use_saved_login():
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)
        
        # 加载之前保存的登录状态
        context = browser.new_context(
            storage_state="jimeng_auth.json",
            viewport={'width': 1280, 'height': 720}
        )
        page = context.new_page()
        
        # 直接访问创作页，无需再次登录
        page.goto("https://jimeng.jianying.com/ai-tool/generate")
        page.wait_for_load_state("networkidle")
        
        print("✅ 已使用保存的登录状态进入即梦")
        
        # 你的业务逻辑...
        
        browser.close()

if __name__ == "__main__":
    use_saved_login()
```

## 四、关键节点说明

| 关键节点 | 说明 | 注意事项 |
|----------|------|----------|
| **官网地址** | `https://jimeng.jianying.com/` | 确保使用 HTTPS |
| **授权域名** | `open.douyin.com` | 抖音 OAuth 授权页面 |
| **登录成功标志** | URL 包含 `jimeng.jianying.com/ai-tool/generate` | 进入创作界面即成功 |
| **二维码定位** | 一般在 `.qr-content img` 或 `img[class*="qrcode"]` | 元素选择器可能需要根据实际页面调整 |
| **登录有效期** | 一般 7～30 天 | 建议保存 storage_state 复用 |

## 五、常见问题

### 1. 二维码加载不出来？
- 检查网络连接
- 增加等待时间：`page.wait_for_timeout(3000)`
- 检查是否被反爬虫机制拦截

### 2. "同意"按钮点不到？
- 按钮可能在 iframe 中，需要切换 frame
- 使用 `force=True` 参数强制点击

### 3. 登录后还是显示未登录？
- 抖音授权后可能有延迟
- 建议等待 3～5 秒再检查页面状态
- 刷新页面确认登录状态

### 4. 如何绕过扫码？
- **即梦目前只支持抖音扫码登录**
- 无法使用账号密码登录
- 必须通过手机抖音 APP 扫码
